# Max2_Pong
## Pin-Pong game optimised for MAX II CPLD (epm240)

One of my first digital logic projects developed in 2022 for a custom DIY MAX II development board.

The primary objective was to implement optimized realisation of game logic and VGA driver for limited number of LEs (240 LEs). Additionally, the project served as a test for the development board.

The project consists of 800x600 60Hz VGA driver and game logic modules.
